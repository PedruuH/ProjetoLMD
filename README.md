
## Pré-requisitos

Java Runtime Environment, Java SE Development Kit e Maven.

### Instalação

```
$ sudo apt update
```
```
$ sudo apt install default-jre
```
```
$ sudo apt install default-jdk
```
```
$ sudo apt install maven
```

## Execução:
```
$ mvn install
```

```
$ mvn exec:java -Dexec.mainClass="com.ufu.ProjetoLMD.ProjetoLMD"
```
